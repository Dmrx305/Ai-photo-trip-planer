# ui-ux-pro-max

## Status

- installed
- available for this project
- assigned to the frontend agent

## Installation

Installed via:

```bash
npx skills add https://github.com/nextlevelbuilder/ui-ux-pro-max-skill --skill ui-ux-pro-max --yes --global
```

Installed location:

- `~/.agents/skills/ui-ux-pro-max`

## Purpose In This Project

The `ui-ux-pro-max` skill extends the frontend workflow with broader UI/UX
reasoning than a pure visual styling skill.

It is useful for:

- interface structure
- accessibility and interaction quality
- layout and spacing standards
- color and typography systems
- motion and visual consistency
- higher-level product UI decisions

## Why It Fits This Project

AI Photo Trip Planner is not only a technical app. It also needs to feel clear,
guided, and trustworthy while still looking creative and photography-oriented.

This skill fits well when the task is about:

- improving usability, not just visuals
- reviewing UI quality before release
- shaping a stronger design system direction
- making the planning experience feel more polished

## Recommended Usage

Use this skill for:

- UX reviews
- accessibility-driven UI cleanup
- component consistency checks
- layout and hierarchy improvements
- design system or token decisions
- more advanced frontend redesign work
